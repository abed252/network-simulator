# Network Simulator (Python)

A networking simulation project implemented in Python.

This project simulates core networking behavior such as packet handling, message flow, and protocol-level logic according to the assignment specification.

---

## Project Structure

- `main.py` –   Core simulation logic  
- `simulator` – Entry point of the program
- `makefile` – Build / execution helper  

---

## Requirements

- Python 3.x
- Linux / macOS / WSL (recommended)

---

## Run

Using Python directly:

```bash
python3 main.py [ARGS]
```

Or using make (if defined in makefile):

```bash
make
```

---

## Description

The simulator models a network environment and processes input according to the defined protocol rules.

Typical responsibilities include:

- Parsing input
- Simulating packet/message transmission
- Applying protocol logic
- Handling events or state transitions
- Producing formatted output according to assignment requirements

---

## Output

The program prints simulation results based on the given input and protocol rules.

Output format strictly follows the assignment specification.
